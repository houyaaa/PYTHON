# 1️⃣ 학습 내용 정리

## 1. 판다스 자료구조 소개

### 개념정리

#### 1. Series

<br>

- Series는 pandas의 기본 자료 구조
- 1차원 배열 형태의 데이터
- 인덱스(index)와 값(value)으로 구성
- NumPy배열과 유사 but 인덱스를 통해 데이터 접근 가능
- 딕셔너리나 리스트를 이용해 생성 가능
- 연산 시 인덱스를 기준으로 자동 정렬


<br>
<br>

#### 2. DataFrame
<br>

- DataFrame은 pandas의 핵심 구조
- 2차원 테이블 형태의 데이터
- 행(index)과 열(column)으로 구성
- 딕셔너리, 리스트, NumPy 배열 등을 이용해 생성 가능
- 엑셀, SQL 테이블과 유사한 구조
- `.T` 를 추가하여 데이터 전치 가능 



<br>
<br>

#### 3. 색인 객체 

<br>

- DataFrame에서 데이터 선택은 매우 중요한 기능
- `loc` 은 라벨 기반 인덱싱
- `iloc` 은 위치 기반 인덱싱
- 특정 행, 열, 조건을 이용해 데이터 선택 가능
- `sort_values()`: 컬럼 기준 정렬 가능
- 오름차순: `ascending = True` or 언급x
- 내림차순: `ascending = False`
<br>
<br>

<br>

 - `iloc`: 인덱싱 끝 점 포함 x

## 2. 핵심 기능
<br>

### 개념정리
<br>
<br>
<br>
<br>
<br>
<br>

#### 1. 재색인 및 삭제 
- 재색인: `reindex` 매서드 호출, 데이터를 새로운 색인에 맞게 재배열함
- 보간: 시계열 데이터 등의 누락된 값은 `method="ffill"`옵션을 통해 직전의 값으로 채워넣기 가능
- 행/열 삭제: `drop` 메서드를 사용하면 선택한 행, 열이 삭제된 새로운 객체를 반환함
   - 열 삭제 시) `columns` 키워드 or `axis=1`을 인수로 넘겨줌 
<br>
<br>
<br>

#### 2. 데이터 선택 및 필터링
- 안전한 선택: 직접 [] 를 사용하여 슬라이싱하면 정수 색인 처리 방식이 헷갈릴 수 있음
- 특수 연산자인 `loc` `iloc` 을 사용하는 것을 권장
- 연쇄 색인 주의: 값을 할당할 떄 색인을 연결해서 사용하는 연쇄색인을 피하고 단일 `loc` 연산을 사용해야 경고없이 원본 데이터가 올바르게 수정됨
- 조건 필터링: 스칼라 비교를 통해 생성된 불리언 배열을 사용하여 조건에 맞는 값을 걸러내기 가능

<br>
<br>
<br>

#### 3. 연산, 함수적용, 정렬
- 산술연산: 서로 다른 색인을 가진 객체를 더하면 결과에 두 색인 통합, 겹치지 않는 부분 -> `NaN`
- 결측치 채우기: `add` 등의 산술 연산 메서드에 `fill_value` 인수를 전달하여 누락된 값을 특수한 값으로 대체 후 계산 가능
- 함수 매핑: DataFrame 의 `apply`메서드를 사용하면 각 행이나 열의 1차원 배열에 일괄적으로 함수를 적용 가능
- 정렬: `sort_index` 는 색인을 알파벳 순으로 정렬 / `sort_values` 는 객체의 데이터 값에 따라 정렬

## 3. 기술 통계 계산과 요약

### 개념정리
<br>
<br>
<br>

판다스는 누락된 데이터(NaN)를 처음부터 제외하고 요약 통계를 계산하도록 설계 -> 편리 
<br>
<br>

#### 1. 기본 오약 통계
- 단일 값 계산: `sum`. `mean`
- Na 처리: 기본적으로 누락된 값을 제외하지만, `skipna=False`옵션 -> NA 값을 그대로 포함여 계산
- 종합요약: `describe` -> 개수, 평균, 최솟값, 최댓값, 사분위수

<br>

#### 2. 상관관계 및 데이터 빈도 측정
- 상관관계와 공분산: `corr` -> 연속하는 두 series 데이터 간의 상관관계 / `cov` 공분산
- 유일값 추출: `unique` 중복되는 값 제거, 유일한 값만 담은 배열
- 도수 계산: `value_counts` 각 데이터 값의 출현 횟수(도수) 계산, 내림차순 정렬
- 멤버십 확인: `isin` 어떤 특정 값이 series에 존재하는지 여부를 불리언 벡터로 쉽게 확인
